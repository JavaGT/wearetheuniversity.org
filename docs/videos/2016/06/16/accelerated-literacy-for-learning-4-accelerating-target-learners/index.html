Description

The Accelerated Literacy for Learning programme is an intervention strategy delivered in Hampton Hill School used a range of techniques to accelerate the progress of target learners.

In this video we examine how acceleration is a key concept that needs to be understood by leaders and teachers, and how the Accelerating Literacy for Learning contract has been framed to allow leaders and teachers to explore this through an inquiry process.
