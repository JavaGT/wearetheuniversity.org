Description

The Accelerated Literacy for Learning programme is an intervention strategy delivered in Hampton Hill School used a range of techniques to accelerate the progress of target learners.

In this video we examine how focused use of professional learning time is a key strategic resourcing tool, helping to build a culture where professional learning time is centred around collective responsibility.
