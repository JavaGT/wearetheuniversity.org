Description

Do you have an idea for a business or project you want to get off the ground?

You could secure $5000 to make it happen with the University of Waikato Summer Start-Up Programme.

Visit http://www.waikato.ac.nz/research/scholarships/SummerStartUp.shtml for more info and to apply – applications close Friday 2 October.
