Student and applicant online services unavailable this weekend

As part of the essential maintenance for the University’s PeopleSoft applications, a project is underway to upgrade Campus Solutions to the latest version of PeopleTools this weekend.

As a result the following systems will be unavailable from 5pm Friday 23 September to 8am Monday 26 September:

  • Student Services Online
  • Application for Admission
  • Expression of Interest
  • ID Cards
  • Request Official Transcript

You will be able to access these systems again on Monday morning (26 September).

Thank you for your patience.
